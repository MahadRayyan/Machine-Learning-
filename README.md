🚀 What This Repo Covers

Real-world messy datasets — cleaning, imputing, and transforming raw data
Exploratory Data Analysis (EDA) with visualizations
Feature engineering and selection techniques
Supervised & unsupervised ML algorithms
Model evaluation, tuning, and comparison
Daily incremental learning — one notebook per day


🧠 The Process
Every day I pick a concept, apply it to a real dataset, and push the notebook here. The goal isn't just to follow tutorials — it's to actually get my hands dirty with messy data, debug broken code, and understand why things work, not just that they work.
This repo is a living document of that process, mistakes and all.
